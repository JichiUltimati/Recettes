Les Recettes sont triées par dossier et par tag.

Pour voir toutes les recettes d'un dossier appuyer sur le nom du dossier dans l'explorateur.

Pour la liste des tags, c'est [ici](https://jichiultimati.github.io/Recettes/tags/).

Pour chercher toutes les recettes d'un tag, cherchez \# tag-que-vous-recherchez dans la barre de recherche, sélectionnez n'importe qu'elle recette puis appuyez sur le tag.
